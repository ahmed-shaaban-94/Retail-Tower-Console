## Intent (one paragraph)

A connectors/settings screen for configuring and monitoring data-source integrations
(POS, Data Pulse, file feeds): connection status per connector, configuration entries,
last-sync indicators, and a connector-detail panel. It is where an operator wires up
and supervises the sources that feed the rest of the console.
